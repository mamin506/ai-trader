# AI Trader

A traditional trade bot focused on the U.S. Stock market. This is a personal trading bot project that uses algorithmic trading strategies without AI agents.

## Overview

This project aims to build a systematic trading bot for the U.S. stock market. The focus is on traditional algorithmic trading strategies rather than AI-powered decision making. Future expansion to other asset classes may be considered.

## Development Environment

This project uses VS Code DevContainer for cross-platform development, ensuring a consistent development environment across Windows, Linux, and MacOS.

### Prerequisites

- [Docker](https://www.docker.com/products/docker-desktop) installed on your system
- [Visual Studio Code](https://code.visualstudio.com/) installed
- [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers) for VS Code

### Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/mamin506/ai-trader.git
   cd ai-trader
   ```

2. Open the project in VS Code:
   ```bash
   code .
   ```

3. When prompted, click "Reopen in Container" or use the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`) and select "Dev Containers: Reopen in Container"

4. VS Code will build the development container and set up the environment automatically

### Container Specifications

- **Base Image**: Ubuntu 24.04
- **Python Version**: 3.12 (system default for Ubuntu 24.04)
- **Development Tools**: Git, essential build tools, and Python development dependencies
